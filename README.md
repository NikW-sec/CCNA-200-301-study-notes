## How I made them

**Days 1–18:** Written manually while watching each lecture.

**Days 19–63:** AI-assisted workflow —
1. I wrote a prompt for Claude (included in this repo) so that whenever I paste a lecture transcript, it generates notes matching my  style. To reduce hallucination, I explicitly instruct it to reference only the source material and not fill gaps with general knowledge.
2. I watch the lecture myself, fully focused on understanding rather than note-taking.
3. I refine Claude's draft using what I learned and include some general examples
4. The content is reviewed for consistency with the transcript, then uploaded to Notion via Claude's Notion integration.
